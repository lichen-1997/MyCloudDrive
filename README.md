# MyCloudDrive
A Cloud Drive Client based on Qt and COS services. 基于Qt和COS服务的云盘客户端。

Development environment 开发环境：VS2017 + Qt 5.14

Project structure 项目结构：

- config 配置
- helper 工具
- plugins 插件
- fend 前端
  - control 控件
  - ui 界面
- bend 后端
  - dao 数据访问层
  - man 业务接口层
- middle 中间件
  - models 数据模型层
  - signals 信号层
